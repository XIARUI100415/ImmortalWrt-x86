# ImmortalWrt-x86
自编译ImmortalWrt-x86固件 
内置ZeroTier、PassWall、AdGuardHome、迅雷快鸟、UU加速器、FRP、解锁网易云灰色（新版）、AdByby、KMS等插件
本仓库活跃更新 请放心使用！
默认输出Hyper-V的vhdx虚拟磁盘/VMWare Workstation的vmdk虚拟磁盘/用于安装的img镜像
